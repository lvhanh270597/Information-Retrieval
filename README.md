# Information-Retrieval
## Install
* Install pip
*sudo apt update*
*sudo apt install python3-pip*
* Install PyQt5, numpy
*sudo pip3 install PyQt5*
*sudo pip3 install numpy*
* Download and extract this repository
*Run python file: _source/main.py_*
